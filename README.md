# Hirelyst

Hirelyst is a career readiness and job-matching prototype designed to help candidates explore job opportunities, assess their skills, and build resumes.

## Features
- Job matching system
- Candidate dashboard
- Employer dashboard
- Resume builder
- Light and dark mode

## Tech Stack
- HTML
- CSS
- JavaScript
